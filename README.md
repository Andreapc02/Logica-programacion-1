# Lógica de Programación 1

## Descripción

Este proyecto consiste en un programa desarrollado en JavaScript que solicita al usuario tres números mediante prompt al hacer 
clic en un botón. El programa identifica el número mayor, el menor y el número del centro, y muestra los resultados ordenados 
de mayor a menor y de menor a mayor en la consola.

## Funcionalidades

- Solicita 3 números al usuario.
- Identifica el número mayor, el menor y el del centro.
- Muestra los números ordenados de mayor a menor.
- Muestra los números ordenados de menor a mayor.
- Detecta cuando los tres números son iguales.

## Posibles salidas dependiendo los numeros a ingresar
Entradas: 
4, 4, 2.
Salida:
4, 4, 2.
2, 4, 4.
Entradas: 
4, 2, 4.
Salida:
4, 4, 2.
2, 4, 4.
Entradas: 
2, 4, 4.
Salida:
4, 4, 2.
2, 4, 4.
